Atividade 5: Classe "Projeto" com Métodos de Status
    Objetivo: Desenvolver o conceito de estado (status) de um objeto.
    Descrição: Crie uma classe Projeto, que representa um projeto em andamento. A classe deve ter os atributos nome (String), descricao (String), status (String).
    Instruções: Adicione métodos para atualizar o status do projeto (ex.: "Em Andamento", "Concluído", "Cancelado") e para exibir as informações do projeto.

Explicação dos Componentes da Classe
    1 - Atributos: A classe possui três atributos (nome, descricao, e status) que representam as informações principais de um projeto.

    2 - Construtor: Inicializa os atributos nome, descricao e status quando um novo objeto Projeto é criado.

    3 - Getters e Setters: Métodos para acessar e modificar os valores dos atributos.

    4 - Método atualizarStatus: Permite alterar o status do projeto. Esse método recebe um novo status como parâmetro e atualiza o atributo status do projeto, exibindo uma mensagem de confirmação.

    5 - Método exibirInformacoes: Exibe todos os detalhes do projeto, incluindo o nome, a descrição e o status atual.
